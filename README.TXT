Ensure that you have a .env file which contains:
- BT_MERCHANT_ID
- BT_PUBLIC_KEY
- BT_PRIVATE_KEY

open a new terminal, run node app.js
You should find the website by typing localhost:3000 into Google.


---------
For Apple Pay
---------
On github, connected to Render, build command: yarn
start command: node app.js

Must open on the render link in Safari.

Give render your:
- BT_MERCHANT_ID
- BT_PUBLIC_KEY
- BT_PRIVATE_KEY

Need to set the domain as approved in Braintree Gateway
Need to be given an applePay Dev account by the team, which you then put onto you Mac.
Need to put a test card into your wallet on your laptop
